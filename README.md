# ts-react-seeds

This Repository contains the seeds for frontend development using Typescript, React, Tailwind and other libraries.

The Main branch is seed for ts-react-tailwindcss.

View the other branches for more seeds.

## Getting Started

### Prerequisites

This template uses **pnpm** as the package manager. If you don't pnpm install it using
below command

```bash
npm install -g pnpm
```

### How to Run:

1. **Install Dependencies**:

```bash
   pnpm install
```

2. **Start Development Server**:

```bash
   pnpm dev
```

## Custom Scripts

    •	pnpm dev: Start the development server.
    •	pnpm build: Build the project for production.
    •	pnpm lint: Run ESLint to check for linting issues.
    •	pnpm format: Run Prettier to format the code.
